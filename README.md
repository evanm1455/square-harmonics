# square-harmonics
Small python script to visualize the harmonics of a square wave. 

An infinite square wave can be well modeled by the sum of a large number of odd harmonics, as they all have the same behavior in the vicinity of `(2n+1)Ï€`.

Requires matplotlib, numpy, scipy
